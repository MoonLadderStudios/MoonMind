# Task Steps System

Status: Active  
Owners: MoonMind Engineering  
Last Updated: 2026-03-14  

## 1. Purpose

The Task Steps system enables a single `AgentTaskWorkflow` Temporal execution to navigate an ordered sequence of programmatic steps. 

It accomplishes:
- One Execution Workflow representing a single overarching agent session.
- Common top-level context (Runtime, Workspace bounds, Repository).
- Step-specific iteration for distinct skills or distinct prompts.

## 2. Canonical Payload Contract Extension

The workflow input accepts an optional `task.steps` array. 

```json
{
  "task": {
    "instructions": "Overall objective",
    "steps": [
      {
        "id": "step-1",
        "title": "Analyze the problem",
        "instructions": "Optional step-specific instructions"
      },
      {
        "id": "step-2",
        "skill": { "id": "bash_executer" }
      }
    ]
  }
}
```

If omitted, the workflow defaults to an implicit single-step process processing the top-level `instructions`.

## 3. Temporal Execution Loop

The Temporal `AgentTaskWorkflow` implementation explicitly cycles through these steps via its Workflow code.

1. It resolves the full step list.
2. For each step, it executes the target Activity (either resolving to an Agent LLM loop for general instructions, or executing a specific script-backed Skill).
3. It emits explicit Temporal Search Attributes and Heartbeats to update the Mission Control UI.
4. If a step errors, the Workflow gracefully intercepts it and can either retry via standard Temporal Retry Policies, or propagate the failure to halt the remainder of the routine.
5. All step execution outputs stream to scoped blocks on the Mission Control Terminal widget.

## 4. Resilience

Because steps are mapped internally to distinct Activity executions:
- State is naturally durable across worker restarts.
- Artifacts generated by `step-1` are seamlessly available to `step-2` via the shared `temporal-worker-sandbox` volume mount.
- Failed steps can be safely retried without re-running earlier, expensive steps.
