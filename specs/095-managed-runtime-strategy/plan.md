## Implementation Steps

### Step 1: Create `ManagedRuntimeStrategy` ABC

**File**: `moonmind/workflows/temporal/runtime/strategies/base.py`

Define the abstract base class with:
- Abstract: `runtime_id` (property), `default_command_template` (property), `build_command()` (method)
- Concrete with defaults: `default_auth_mode` (returns `"api_key"`), `shape_environment()` (identity pass-through), `prepare_workspace()` (no-op), `classify_exit()` (0=completed, else failed), `create_output_parser()` (returns `None` for Phase 1)

### Step 2: Create `RUNTIME_STRATEGIES` Registry

**File**: `moonmind/workflows/temporal/runtime/strategies/__init__.py`

Module-level dict. Imports and registers `GeminiCliStrategy`. Exports `RUNTIME_STRATEGIES`, `ManagedRuntimeStrategy`, and strategy lookup helper.

### Step 3: Implement `GeminiCliStrategy`

**File**: `moonmind/workflows/temporal/runtime/strategies/gemini_cli.py`

- `runtime_id = "gemini_cli"`
- `default_command_template = ["gemini"]`
- `default_auth_mode = "api_key"`
- `build_command()`: Extract logic from `launcher.py:342-351`
- `shape_environment()`: Pass through `HOME`, `GEMINI_HOME`, `GEMINI_CLI_HOME` from worker env

### Step 4: Wire Launcher Delegation

**File**: `moonmind/workflows/temporal/runtime/launcher.py`

In `build_command()`:
1. Import `RUNTIME_STRATEGIES`
2. Before the `if/elif` block, attempt `strategy = RUNTIME_STRATEGIES.get(profile.runtime_id)`
3. If strategy is found, return `strategy.build_command(profile, request)`
4. Otherwise fall through to existing `if/elif`

### Step 5: Wire Adapter Delegation

**File**: `moonmind/workflows/adapters/managed_agent_adapter.py`

In `start()`:
1. Import `RUNTIME_STRATEGIES`
2. For `command_template` defaults (L286-293): check strategy first
3. For `auth_mode` defaults (L236-238): check strategy first
4. Fall through to existing defaults for unregistered runtimes

### Step 6: Unit Tests

**Files**: `tests/unit/workflows/temporal/runtime/strategies/test_base.py`, `test_gemini_cli.py`

- Test ABC cannot be instantiated
- Test `GeminiCliStrategy.build_command()` produces identical output to existing launcher path
- Test `GeminiCliStrategy.shape_environment()` passes through expected keys
- Test `GeminiCliStrategy` properties return expected defaults
- Test `RUNTIME_STRATEGIES` contains `"gemini_cli"` entry
- Test launcher fallthrough works for unregistered runtimes

### Step 7: Regression Verification

Run `./tools/test_unit.sh` and verify all existing tests pass.
